# 跨机器使用配置示例

## 1. 查看服务机器的真实IP
```bash
# 在运行OCR服务的机器上执行
ip addr show | grep "inet " | grep -v 127.0.0.1
# 输出示例：192.168.1.100
```

## 2. 其他电脑的配置文件修改
```toml
[ocr]
# 改为服务机器的真实IP
server_url = "http://192.168.1.100:8082"
model = "chandra-ocr"
prompt_preset = "default"
```

## 3. 命令行参数方式
```bash
python convert_pdfs_to_md.py --server-url http://192.168.1.100:8082
```

## 4. 网络要求
- 确保两台机器在同一网络
- 防火墙允许8082端口访问
- 服务机器的llama-server保持运行
